Var: It can be accesess within and outside of a block. It can be accesed by whole script. It can be moddified.
Let: It's scope is it's nearest enclosing block. it can be modified within the block
Const: It is also block scope. It can not be re-declared or modified.

Null : It's used to represent no value. when coder want to represent the intentional absence of a value Null is used
Undefined : When a variable is not assigned a value undefined is applied

REST API :
REST API means Representational State Transfer Application Programming Interface. It is used in web and mobile applications to retrieve or modify resources and data on remote systems.Rest is an architectural style for creating websites using the HTTP protocol.
